## The Burnside process for integer partitions, contingency tables and Sylow $p$-double cosets.

An implementation of various examples of the Burnside process.

The file `partitions.jl` implements the reflected and lumped Burnside process for integer partitions [DH25](https://arxiv.org/abs/2503.02818).

The file `contingency_tables.jl` implements the lumped Burnside process for contingency tables [DH25](https://arxiv.org/abs/2503.02818).

The file `sylow_double_cosets.jl` implement the Burnside process for Sylow $p$-double cosets in the symmetric group [H25](https://arxiv.org/abs/2511.01058).

The folder `Examples` contains Jupyter notebooks demonstrating the use of both Markov chains. The notebooks in this folder can be used to recreate the figures in [DH25](https://arxiv.org/abs/2503.02818) and [H25](https://arxiv.org/abs/2511.01058).
